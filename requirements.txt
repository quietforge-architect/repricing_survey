# Node / JS tooling (handled via package.json):
# - express, body-parser, dotenv, better-sqlite3 (runtime deps) -- installed via `npm install`
# - playwright (devDependency) -- installed via `npm i -D playwright`
# - qrcode (devDependency) -- installed via `npm i -D qrcode` for QR splash-card generation
# - lite-server (devDependency) -- installed via `npm i -D lite-server` for live preview with reload

# Notes:
# - Playwright requires additional browser binaries installed via `npx playwright install`.
# - On Windows, native builds (e.g., better-sqlite3) may require Python and Visual C++ Build Tools.
#   Install Visual Studio Build Tools: winget install --id Microsoft.VisualStudio.2022.BuildTools -e
#   or install the "Desktop development with C++" workload.
